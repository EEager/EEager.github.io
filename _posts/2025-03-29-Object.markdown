# **블로그 작성 목적**

블로그를 작성하려는 목적은 **ML(Machine Learning)**과 **AI(Artificial Intelligence)**에 대해 깊이 연구하거나 학술적으로 접근하려는 것이 아니라, 지금까지 개발된 다양한 기술들을 전반적으로 이해하고 이를 직접 실습해보는 데 있음.
주로 **실습 위주로 기술들을 탐구하며 사용해보는 경험**을 기록하기 위한 블로그.

---

# 추가 계획

ML/AI를 공부하는 과정에서 다음과 같은 주제를 함께 다룰 예정:
1. **알고리즘(Algorithm)**
   - 알고리즘 학습 및 구현.
   - ML/AI에서 활용되는 알고리즘 이해.

2. **DS/DE 스터디**
     - 현재 사용 중인 데이터 사이언스(DS)와 데이터 엔지니어링(DE) 기술을 탐구하여 **각 기술의 개요와 역할**을 이해하고, 이를 바탕으로 **현업에서 어떻게 활용되고 있는지 인사이트를 키우는 것**에 중점.
     - DS : 
       - ML 및 AI에 활용되는 DS 기술들의 구현 및 학습. 실제 프로젝트(비즈니스 문제 해결)에 DS 도구를 적용하여 결과 도출.
     - DE : 
       - ML 모델에 필요한 데이터의 준비 단계에서 DE 기술의 역할 이해. 실시간 데이터 처리와 스토리지 최적화 프로세스에 대한 학습.
